## ✨ Возможности

- **⬇ Скачивание модов** через SteamCMD — один мод или целые списки
- **📦 Импорт коллекций Steam** — вставьте ссылку на коллекцию и все моды добавятся сразу
- **🔍 Автоопределение Game ID** — просто вставьте Mod ID, Game ID заполнится сам
- **🔄 Проверка обновлений** — сканирует локальную папку с модами и показывает устаревшие
- **⏸ Пауза и продолжение** — остановите загрузку и продолжите при следующем запуске
- **🔘 Включение / отключение модов** — переключайте моды без удаления (папка переименуется в `.disabled`)
- **📋 История игр** — запоминает все игры для которых вы скачивали моды
- **📁 Открытие папки одним кликом** — прямо из таблицы
- **💾 Столбец с размером** — видно сколько места занимает каждый мод
- **🌐 Локализация** — полный перевод интерфейса через JSON-файлы
- **🖥 Кросс-платформенность** — Windows, Linux, macOS

## 🚀 Быстрый старт

1. Клонируйте или скачайте репозиторий
2. Установите зависимости (см. выше)
3. Запустите:
   ```bash
   python workshopdl.py
   ```
4. Перейдите в **Настройки** → нажмите **⬇ Скачать SteamCMD автоматически**
5. На вкладке **Скачать** введите Mod ID — Game ID определится автоматически
6. Нажмите **⬇ Скачать**

## 🔄 Проверка обновлений

Вкладка **🔄 Проверка обновлений** позволяет сканировать любую локальную папку с модами
(например: `C:\games\SovietRepublic\media_soviet\workshop_wip`).

Папка должна содержать числовые подпапки — по одной на мод:
```
workshop_wip/
├── 1797996358/
├── 1807300910/
└── 2031421793.disabled   ← отключённый мод (игра его игнорирует)
```

WorkshopDL сравнивает дату изменения локальной папки с полем `time_updated`
из Steam API и помечает каждый мод:

| Иконка | Значение |
|---|---|
| 🔴 | Устарел — на сервере есть более новая версия |
| 🟢 | Актуален |
| 🔘 | Отключён (папка имеет суффикс `.disabled`) |
| ⚪ | Неизвестно — Steam API не вернул данных |

## 🔘 Включение / отключение модов

Нажмите кнопку **⏸ / ▶** в таблице чтобы переключить мод.  
Программа просто переименовывает папку:

```
1797996358          →   1797996358.disabled    (отключён)
1797996358.disabled →   1797996358             (включён)
```

Файлы не удаляются. Игра будет игнорировать папки с суффиксом `.disabled`
(зависит от загрузчика модов конкретной игры).

## 🌐 Переводы

Все строки интерфейса хранятся в одном JSON-файле. Чтобы создать новый перевод:

1. Скопируйте `lang_en.json` и переименуйте, например в `lang_fr.json`
2. Переведите **значения** (правая часть каждой строки) — ключи менять не нужно
3. В **Настройках → Язык** укажите путь к файлу и нажмите **✅ Применить**

### Поделиться переводом

Чтобы ваш перевод был доступен всем пользователям:
- Добавьте файл `lang_XX.json` в папку `lang/` этого репозитория
- Откройте Pull Request — мы добавим его в список загрузки внутри программы

## ⚙ Настройки

| Настройка | Описание |
|---|---|
| Анонимный режим | Скачивание без аккаунта Steam (большинство модов поддерживает) |
| Логин / пароль Steam | Нужны только если анонимный режим выключен |
| Путь к SteamCMD | Путь к бинарю `steamcmd` — или скачайте автоматически |
| Язык | Путь к файлу локализации `.json` |
| Папка модов | Папка по умолчанию для проверки обновлений |

## 🛠 SteamCMD

WorkshopDL использует [SteamCMD](https://developer.valvesoftware.com/wiki/SteamCMD)
для скачивания модов. Устанавливать вручную не нужно —
перейдите в **Настройки** и нажмите **⬇ Скачать SteamCMD автоматически**.

При первом запуске SteamCMD скачивает собственные файлы движка (~40 МБ).
Это происходит один раз и отображается в логе настроек.

| Платформа | Бинарь | Архив |
|---|---|---|
| Windows | `steamcmd.exe` | `.zip` |
| Linux | `steamcmd.sh` | `.tar.gz` |
| macOS | `steamcmd` | `.tar.gz` |
